⚙️ CONFIGURACIÓN DE VOZ GLOBAL

Para alternar el motor del sistema entre voces de hombres o mujeres:

  • Android: Ingrese a Ajustes ➔ Sistema (o Administración General) ➔ Idioma e introducción ➔ Síntesis de voz. Toque el icono de engranaje para descargar variantes de idiomas y elija los perfiles numéricos de voz preferidos.
  • iOS / Apple: Diríjase a Ajustes ➔ Accesibilidad ➔ Contenido leído ➔ Voces. Busque el idioma deseado y descargue las versiones extendidas de alta fidelidad para seleccionar géneros específicos.
